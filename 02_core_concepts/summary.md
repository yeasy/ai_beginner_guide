# 第二章：AI 核心概念速览

## 本章小结

本章系统介绍了 AI 领域的核心概念，帮助读者建立了扎实的概念基础。

### 核心要点回顾

**AI、机器学习与深度学习的关系**
- 三者是包含关系：AI ⊃ 机器学习 ⊃ 深度学习
- AI 是追求让机器表现智能的广泛领域
- 机器学习是实现 AI 的主要方法，通过数据学习规律
- 深度学习使用多层神经网络，是当前最先进的 AI 技术

**数据：AI 的燃料**
- 数据质量和数量决定模型的上限
- 数据类型包括结构化、非结构化和半结构化
- 数据集由样本、特征和标签组成
- 数据质量需关注准确性、完整性、一致性、时效性和代表性
- 数据标注是监督学习的关键环节，成本高昂

**算法与模型**
- 算法是从数据中学习的方法，模型是算法学习的成果
- 机器学习算法分为分类、回归、聚类、降维等类型
- 神经网络由输入层、隐藏层和输出层组成
- 模型参数在训练中学习，超参数需人工设定
- 常见架构包括 CNN、RNN、Transformer 等

**训练、推理与评估**
- 训练是模型从数据中学习知识的过程
- 推理是使用训练好的模型进行预测
- 评估指标包括准确率、精确率、召回率、F1 等
- 过拟合是训练好但测试差，欠拟合是训练和测试都差
- 模型部署后需要持续监控和更新

### 关键术语

| 术语 | 解释 |
|------|------|
| 机器学习 (ML) | 让计算机从数据中自动学习规律的方法 |
| 深度学习 (DL) | 使用多层神经网络的机器学习分支 |
| 监督学习 | 从带标签的数据中学习 |
| 无监督学习 | 从无标签的数据中发现模式 |
| 强化学习 | 通过与环境交互获得反馈来学习 |
| 神经网络 | 模拟人脑结构的计算模型 |
| 训练 | 模型从数据中学习的过程 |
| 推理 | 使用模型进行预测的过程 |
| 损失函数 | 量化预测与真实值差距的函数 |
| 过拟合 | 模型在训练集上好但在测试集上差 |
| 欠拟合 | 模型在训练集上就表现不好 |

### 关键公式

| 指标 | 公式 | 用途 |
|------|------|------|
| 准确率 | (TP+TN)/(TP+TN+FP+FN) | 整体正确率 |
| 精确率 | TP/(TP+FP) | 预测为正中真正为正的比例 |
| 召回率 | TP/(TP+FN) | 实际为正中被预测为正的比例 |
| F1 分数 | 2×P×R/(P+R) | 精确率和召回率的调和平均 |
