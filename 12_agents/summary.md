## 本章小结

本章我们解开了 AI 进化的下一个形态——智能体（Agent）。

### 核心要点回顾

1. **从“说”到“做”**：智能体不仅是大脑，还装上了感官（感知）和手脚（工具），能主动改变世界。
2. **ReAct 模式**：智能体像人一样，遵循“思考-行动-观察-再思考”的逻辑闭环。
3. **多智能体协作**：让 AI 扮演不同角色（产品、开发、测试），组成虚拟团队解决复杂问题。
4. **工程化是分水岭**：评测（Evals）、防护（Guardrails）、可观测（Observability）决定智能体能否从 Demo 走向生产。
5. **人人可造智能体**：通过 Coze、Dify 等平台，普通人不需要写代码，通过拖拽流程图就能开发自己的 AI 应用。

### 关键术语

| 术语 | 英文 | 解释 |
| :--- | :--- | :--- |
| **智能体** | Agent | 具备感知、规划、行动能力的 AI 系统 |
| **ReAct** | Reasoning + Acting | 推理与行动交替的思考模式 |
| **工作流** | Workflow | 预定义的一系列任务步骤 |
| **低代码平台** | Low-Code Platform | 通过图形界面开发应用的工具 |
| **MCP** | Model Context Protocol | 让模型更标准化地连接工具与上下文的协议思路 |
| **评测** | Evals | 用任务集系统衡量智能体质量与稳定性 |
