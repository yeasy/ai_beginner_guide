# 第十二章：AI 智能体与自动化

## 12.3 低代码智能体开发平台

看到这里，你可能想："这太高级了，我得是个程序员才能搞吧？"

好消息：现在有很多 **低代码/零代码（Low-Code/No-Code）平台**，让普通人通过拖拉拽，就能造出自己的 Agent。

### 12.3.1 扣子（Coze）

字节跳动推出的超强大平台，国内国外都有版（Coze.cn / Coze.com）。

**特点**：
- **免费**：可以使用 GPT-4、Gemini、云雀等顶级模型。
- **插件丰富**：内置了新闻、天气、甚至抖音、头条的插件。
- **工作流（Workflow）**：像画流程图一样编排 AI 的工作步骤。
- **发布**：一键发布到微信公众号、飞书、掘金等平台。

**实战案例：创建一个"每天自动发新闻早报"的 Bot**
1. **创建 Bot**：起名"早报君"。
2. **编排工作流**：
   - 节点 1：每天早上 8 点触发。
   - 节点 2：调用"搜索新闻"插件，搜"科技热点"。
   - 节点 3：LLM 总结新闻，写成 300 字摘要。
   - 节点 4：调用"微信公众号"插件，发送文章。
3. **测试发布**：搞定！

### 12.3.2 Dify

开源界的明星项目，适合极客和企业用户。

**特点**：
- **开源**：你可以把它部署在自己的电脑或服务器上，数据绝对安全。
- **检索增强生成（RAG）引擎强大**：上传知识库（PDF、Word），它的检索效果非常好。
- **模型中立**：你可以随意切换底层模型（GPT、Claude、文心一言、本地 Ollama）。

适合场景：给公司做一个 **内部知识库问答助手**。

### 12.3.3 Zapier / Make

自动化领域的元老，现在也接入了 AI。

**特点**：连接一切。它可以把 Gmail、Notion、Slack、Trello 等 5000+ 个应用连起来。

**场景**：
- 有新邮件 -> 发给 ChatGPT 总结 -> 把总结发到 Slack 群里。
- Notion 里新增一个待办 -> 同步到 Google 日历。

### 12.3.4 怎么选择？

- **小白入门、玩玩看**：首选 **Coze（扣子）**，简单免费强大。
- **企业内部、极客折腾**：选 **Dify**。
- **连接各种 App**：选 **Zapier**。

**普通人的机会**
未来，"会写代码"可能不再稀缺，但"会设计工作流"（Workflow Design）将成为核心竞争力。你能把 AI 和工具像乐高积木一样搭起来解决问题，你就是超级个体。
