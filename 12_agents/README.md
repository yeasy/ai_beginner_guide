# 第十二章：AI 智能体与自动化

> 从"陪你聊天"到"帮你干活"：AI 进化的下一站

---

如果你觉得只能聊天的 ChatGPT 已经很强了，那接下来的内容可能会让你脑洞大开。

AI 领域正在发生一场更为深刻的革命——从**聊天机器人（Chatbot）**向**智能体（Agent）**进化。

区别在哪里？
- **Chatbot**：你说"我渴了"，它会告诉你"多喝热水"。
- **Agent**：你说"我渴了"，它会**操控**机器人给你倒杯水端过来，或者**调用**外卖软件帮你点杯奶茶。

智能体不仅能"说"，还能"做"。它们能感知环境、规划步骤、使用工具，甚至与其他智能体协作，最终完成一个复杂的自动化任务。

本章将带你走进智能体的世界，看看这些"数字打工人"是如何工作的，以及作为普通人，我们如何通过低代码平台组建自己的"AI 团队"。
