# 第十一章：AI 实战应用场景

## 11.3 编程辅助：不懂代码也能开发工具

你可能完全不懂编程，但不代表你不能用代码解决问题。AI 就是你的程序员。

### 场景 1：文件批量重命名

**痛点**：文件夹里有 100 张照片，文件名乱七八糟，想改成"旅游_001.jpg", "旅游_002.jpg"...

**提示词模板**：
> "我不是程序员，但我电脑上装了 Python。
> 我有个文件夹路径是 `D:\Photos`。
> 里面有很多 `.jpg` 图片。
> 请帮我写一段 Python 代码，把它们按修改时间排序，重命名为 'Travel_001.jpg', 'Travel_002.jpg' 这种格式。
> 告诉我怎么运行这段代码。"

你只需要复制它给的代码，保存成文件，运行，搞定。

### 场景 2：数据自动化清洗

**痛点**：有个 Excel 表，里面全是"张三（13800138000）"这种格式，想把姓名和电话分到两列。

**提示词模板**：
> "我有一列数据，格式是 '姓名（手机号）'。
> 请给我写一个 Excel 公式，或者一段 Python 脚本。
> 帮我把姓名提取到 A 列，手机号提取到 B 列，去掉括号。"

### 场景 3：做一个简单的个人主页

**痛点**：想给自己做一个简单的展示网页，放简历和作品。

**提示词模板**：
> "请帮我写一个简单的个人主页 HTML 文件。
> 风格：**极简主义，苹果风**。
> 包含内容：
> 1. 圆形头像（用占位符）。
> 2. 名字：王小明。
> 3. 简介：设计师 & 摄影师。
> 4. 三个链接按钮：微博、作品集、联系我。
> 请给我完整的 HTML 和 CSS 代码，我直接保存就能看。"

配合 Claude 的 Artifacts 功能，你可以实时看到网页效果，还能让它"把按钮改成圆角"、"背景换成浅灰色"。

### 场景 4：解释看不懂的报错

**痛点**：运行软件或代码时弹出一串英文报错，看不懂。

**提示词模板**：
> "我的电脑弹出了这个错误：
> `Error: EACCES: permission denied, open '/Users/name/config.json'`
> 这是什么意思？我该如何在 Mac 上解决这个问题？请用**小白能懂的大白话**解释。"

它会告诉你："这是权限不足的意思，你试着在命令前加个 `sudo`..."
