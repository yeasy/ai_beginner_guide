# 第七章：多模态与生成式 AI

## 7.2 图像生成技术

AI 图像生成是生成式 AI 最引人注目的应用之一。从文字描述生成高质量图像，这种能力曾被认为是人类独有的创造力。

### 7.2.1 图像生成的发展

**GAN 时代（2014-2020）**

生成对抗网络（GAN）是早期图像生成的主流方法：
- 生成器和判别器相互对抗
- 可以生成逼真的人脸
- 但训练不稳定，难以控制

**扩散模型时代（2020-至今）**

扩散模型成为新的主流：
- 基于逐步去噪的过程
- 训练更稳定
- 生成质量大幅提升

### 7.2.2 扩散模型原理

**前向过程（加噪）**

逐步向图像添加噪声，直到成为纯噪声：
```
原始图像 → 轻微噪声 → 更多噪声 → ... → 纯噪声
```

**反向过程（去噪）**

学习从噪声中恢复图像：
```
纯噪声 → 去噪 → 去噪 → ... → 清晰图像
```

**条件生成**

通过文本条件引导生成过程：
- 文本编码为向量
- 在每步去噪时注入文本信息
- 生成与文本描述匹配的图像

### 7.2.3 主流图像生成模型

**DALL-E（OpenAI）**
- DALL-E 3：最新版本，与 ChatGPT 集成
- 擅长遵循复杂提示
- 商业化程度高

**Midjourney**
- 艺术风格突出
- 通过 Discord 使用
- 社区活跃

**Stable Diffusion**
- 完全开源
- 可本地运行
- 丰富的社区扩展

**Adobe Firefly**
- 专注商业可用性
- 与 Adobe 生态整合
- 版权问题更清晰

### 7.2.4 关键技术要素

**文本编码器**

将提示词转换为向量表示：
- CLIP Text Encoder
- T5 Encoder
- 决定了模型对提示的理解能力

**去噪网络（U-Net）**

实际执行图像生成：
- 预测每步要去除的噪声
- 典型使用 U-Net 架构
- 参数量决定生成能力

**采样器**

控制去噪过程：
- DDPM、DDIM、DPM-Solver 等
- 影响速度和质量的平衡

### 7.2.5 图像生成的应用

**创意设计**
- 概念艺术和插画
- 广告素材生成
- 游戏和影视概念设计

**电商场景**
- 商品图片生成
- 场景图和效果图
- 批量素材生产

**个人创作**
- 头像和壁纸
- 社交媒体配图
- 个性化内容

### 7.2.6 局限与挑战

**准确性问题**
- 手指、文字常出错
- 复杂空间关系难以把握
- 细节一致性问题

**版权争议**
- 训练数据的版权问题
- 生成内容的归属
- 艺术家权益保护

**滥用风险**
- 深度伪造（Deepfake）
- 虚假信息传播
- 需要检测和水印技术
