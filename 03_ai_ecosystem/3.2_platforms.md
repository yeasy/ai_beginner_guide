# 第三章：AI 技术生态与工具全景

## 3.2 主流 AI 平台与服务

AI 技术的商业化催生了丰富的平台和服务生态。无论是希望快速接入 AI 能力的开发者，还是希望使用 AI 提升效率的企业，都有多种选择。

### 3.2.1 云厂商的 AI 服务

三大云厂商都提供全面的 AI 服务：

**亚马逊 AWS**

- **Amazon SageMaker**：全托管 ML 平台，覆盖训练到部署全流程
- **Amazon Bedrock**：接入多家基础模型的统一服务
- **Amazon Rekognition**：图像和视频分析
- **Amazon Polly/Transcribe**：语音合成和识别
- **Amazon Comprehend**：自然语言处理

**微软 Azure**

- **Azure Machine Learning**：企业级 ML 平台
- **Azure OpenAI Service**：托管的 OpenAI 模型服务
- **Azure Cognitive Services**：开箱即用的 AI API
- **GitHub Copilot**：AI 编程助手

**Google Cloud**

- **Vertex AI**：统一的 ML 平台
- **Gemini API**：Google 最新的多模态模型
- **AutoML**：自动化机器学习
- **Cloud Vision/Natural Language**：视觉和语言 API

**国内云厂商**

- **阿里云**：PAI 平台、通义系列模型
- **腾讯云**：TI 平台、混元大模型
- **华为云**：ModelArts、盘古大模型
- **百度智能云**：飞桨平台、文心系列模型

### 3.2.2 大语言模型服务

2023-2026 年，大语言模型（LLM）成为 AI 服务的核心：

**OpenAI**

- 产品：ChatGPT、GPT-4、DALL-E、Sora
- 特点：能力领先，生态最完善
- 访问方式：网页、App、API
- 定价：按 token 计费，有免费额度

**Anthropic**

- 产品：Claude 系列（Claude 3.5 Sonnet/Haiku/Opus）
- 特点：安全性强，上下文窗口大（可达 200K tokens）
- 访问方式：网页、API
- 定价：按 token 计费

**Google**

- 产品：Gemini（Ultra/Pro/Nano）
- 特点：多模态能力强，与 Google 生态整合
- 访问方式：Gemini 应用、Google AI Studio、Vertex AI
- 定价：有免费额度，按量计费

**国内大模型**

| 厂商 | 模型 | 特点 |
|------|------|------|
| 百度 | 文心一言 | 中文理解强，知识库丰富 |
| 阿里巴巴 | 通义千问 | 开源版本质量高 |
| 智谱 | ChatGLM | 学术背景，开源友好 |
| 月之暗面 | Kimi | 长上下文能力突出 |
| 字节跳动 | 豆包 | 与抖音生态整合 |

### 3.2.4 AI 工具选择建议

面对众多选择，如何做出合适的决策？

**按需求场景**

| 需求 | 推荐方案 |
|------|----------|
| 日常问答和写作 | ChatGPT、Claude、Gemini |
| 企业级部署 | Azure OpenAI、AWS Bedrock、Vertex AI |
| 图像生成 | Midjourney（艺术）、DALL-E（通用） |
| 代码开发 | GitHub Copilot、Cursor |
| 中文场景 | 通义千问、文心一言、Kimi |

**考虑因素**

1. **能力需求**：哪些功能是必须的？
2. **成本预算**：API 按量计费 vs 订阅制
3. **数据安全**：敏感数据的处理要求
4. **合规要求**：是否需要数据本地化
5. **集成复杂度**：与现有系统的对接成本

**混合策略**

许多企业采用混合策略：
- 核心业务使用私有化部署
- 边缘场景使用云端 API
- 多个供应商互为备份

AI 服务市场变化迅速，保持关注和定期评估是必要的。
