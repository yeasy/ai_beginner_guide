# 第八章：主流 AI 工具使用指南

## 8.4 国产 AI 工具使用指南

对于国内用户来说，使用国产 AI 工具有天然优势：无需翻墙、中文支持好、符合数据合规要求。本节详细介绍几款最受欢迎的国产 AI 产品。

### 8.4.1 文心一言：百度的全能助手

**产品特点**

文心一言是百度推出的 AI 对话助手，背后是 ERNIE 系列大模型。它的核心优势在于与百度知识图谱的深度整合，在事实性问题上表现出色。

**如何使用**

1. **网页版**：访问 `yiyan.baidu.com`，用百度账号登录即可
2. **App**：在应用商店搜索"文心一言"下载
3. **API**：通过百度智能云 `qianfan.baidubce.com` 接入

**适合场景**

- **知识问答**：充分利用百度搜索和知识图谱
- **中文写作**：公文、报告、文案创作
- **代码辅助**：支持多种编程语言
- **图片理解**：上传图片进行分析

**使用技巧**

```
示例提示词：
"请帮我写一份关于2025年新能源汽车市场的分析报告，
要求包含市场规模、主要玩家、发展趋势三个部分，
每部分500字左右，语言正式专业。"
```

### 8.4.2 通义千问：阿里的智能伙伴

**产品特点**

通义千问是阿里巴巴达摩院推出的 AI 助手。它有一个独特优势：同名模型 Qwen 系列在开源社区质量顶尖，这意味着背后技术实力过硬。

**如何使用**

1. **网页版**：访问 `tongyi.aliyun.com`
2. **App**：应用商店搜索"通义千问"
3. **钉钉集成**：可在钉钉内直接 @通义 使用
4. **API**：通过阿里云 DashScope 服务接入

**核心能力**

- **多模态**：支持文本、图像、音频、视频
- **文档分析**：上传 PDF、Word 等文件进行分析
- **代码生成**：基于 Qwen-Coder 的强大代码能力
- **长文本**：支持超过十万字的长文档

**特色功能**

- **通义万相**：AI 绘画，输入文字生成图片
- **通义听悟**：音视频转文字，支持会议纪要
- **通义灵码**：代码助手，IDE 插件形式

**使用技巧**

```
示例提示词：
"我上传的这份合同有哪些潜在风险点？
请逐条列出，并说明可能的影响和建议的修改方式。"
```

### 8.4.3 DeepSeek：性价比之王

**产品特点**

DeepSeek 是近年崛起的黑马，以极低的成本训练出接近顶级闭源模型性能的模型而闻名。它的推理版本（R1）在复杂逻辑任务上表现尤其出色。

**如何使用**

1. **网页版**：访问 `chat.deepseek.com`
2. **App**：应用商店搜索"DeepSeek"
3. **API**：访问 `platform.deepseek.com` 获取密钥

**核心优势**

- **价格极低**：API 价格可能是 GPT-4 的十分之一
- **推理能力强**：R1 版本擅长数学、逻辑、代码
- **响应快速**：模型推理效率高
- **开源透明**：核心模型开源，技术论文公开

**版本选择**

| 版本 | 适用场景 |
|------|----------|
| DeepSeek-V3 | 通用对话、写作、翻译 |
| DeepSeek-R1 | 数学推理、逻辑分析、复杂编程 |
| DeepSeek-Coder | 代码生成、代码解释 |

**使用技巧**

当遇到复杂问题时，可以使用 R1 版本并观察它的"思考过程"：

```
示例提示词：
"一个水池有两个进水管和一个出水管。
单开第一个进水管6小时灌满，
单开第二个进水管4小时灌满，
单开出水管8小时放完。
如果三管齐开，多久能灌满水池？
请详细展示推理过程。"
```

### 8.4.4 Kimi：长文档处理专家

**产品特点**

Kimi 是月之暗面推出的 AI 助手，以超长上下文著称。当你需要处理长论文、分析财报、阅读整本书时，Kimi 是首选。

**如何使用**

1. **网页版**：访问 `kimi.moonshot.cn`
2. **App**：应用商店搜索"Kimi"
3. **浏览器插件**：可在浏览时快速总结网页

**核心优势**

- **超长上下文**：支持最高 200 万字
- **联网搜索**：可获取最新信息
- **文件分析**：支持 PDF、Word、TXT 等
- **网页解析**：输入链接自动提取内容

**典型使用场景**

1. **论文阅读**
   ```
   上传一篇PDF论文，问："这篇论文的核心创新点是什么？
   实验设计有哪些不足？"
   ```

2. **财报分析**
   ```
   上传公司年报，问："这家公司的主营业务收入构成是什么？
   同比变化如何？有哪些风险因素？"
   ```

3. **多文档对比**
   ```
   上传多份竞品分析报告，问："这三个产品的核心差异是什么？
   各自的优缺点？"
   ```

### 8.4.5 豆包：字节跳动的多面手

**产品特点**

豆包是字节跳动推出的 AI 助手，特点是多模态能力强、与抖音生态整合紧密、价格亲民。

**如何使用**

1. **App**：应用商店搜索"豆包"（主推方式）
2. **网页版**：访问 `doubao.com`
3. **开发者**：通过火山引擎 API 接入

**核心功能**

- **AI 对话**：日常问答、写作辅助
- **图片生成**：文生图功能
- **智能体**：可创建定制化 AI 角色
- **语音对话**：支持实时语音交互

**特色体验**

豆包 App 提供了丰富的"智能体"，类似于预设好特定人设和能力的 AI 角色：
- 英语口语陪练
- 小红书文案助手
- 面试模拟官
- 心理咨询师

### 8.4.6 智谱清言：学术背景的可靠选择

**产品特点**

智谱清言是智谱 AI（清华系创业公司）推出的产品，学术背景深厚，在专业领域表现稳定可靠。

**如何使用**

1. **网页版**：访问 `chatglm.cn`
2. **App**：应用商店搜索"智谱清言"
3. **API**：访问 `open.bigmodel.cn` 接入

**核心优势**

- **专业可靠**：学术背景，输出相对严谨
- **代码能力**：CodeGeeX 系列代码模型能力强
- **企业服务**：提供成熟的企业级解决方案
- **开源贡献**：ChatGLM 系列开源，便于研究

### 8.4.8 使用国产工具的注意事项

**优势**

1. **访问便利**：无需特殊网络环境
2. **中文优化**：针对中文场景深度优化
3. **数据合规**：符合国内数据监管要求
4. **本土生态**：与国内应用场景紧密结合
5. **价格亲民**：竞争激烈，价格普遍较低

**注意点**

1. **功能边界**：了解各工具的能力边界，不要强求
2. **隐私意识**：敏感信息谨慎输入，了解数据使用政策
3. **多工具配合**：不同任务用不同工具，发挥各自优势
4. **保持学习**：国产 AI 发展迅速，持续关注新功能

**小建议**

刚开始使用时，建议先选择 1-2 个工具深度体验：
- **通用使用**：试试通义千问或豆包
- **技术向**：试试 DeepSeek
- **学术向**：试试 Kimi

熟练后再根据具体场景切换工具，做到"术业有专攻"。
