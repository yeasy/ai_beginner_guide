# 第八章：主流 AI 工具使用指南

## 8.5 其他实用 AI 工具

除了前面介绍的通用对话 AI 工具，在特定领域还有很多身怀绝技的专业工具。这里挑选几个最适合普通人上手、好玩又实用的。

### 8.5.1 效率办公与写作 (Productivity)

这是普通人最能感受到 AI 提效的领域。

**国际首选：Notion AI**
- **特点**：将 AI 植入笔记文档，直接在文档里帮你扩写、润色、总结、列待办。
- **2026 状态**：已成为全球标准，支持自动整理整个知识库。

**国产替代：飞书智能伙伴 (Feishu My AI) / 钉钉魔法棒**
- **飞书**：擅长会议纪要转待办、多语种实时翻译、根据文档自动生成数据图表。
- **WPS AI**：如果你还在用 Word/PPT，它是最佳选择。一键美化 PPT、让 Excel 自动写公式。

### 8.5.2 搜索与深度研究 (Search & Research)

告别传统的"十个蓝色链接"，直接获取答案。

**国际首选：Perplexity / SearchGPT**
- **特点**：实时联网，阅读数十个网页后给你一份带引用的"研究报告"。
- **2026 状态**：已基本取代传统搜索引擎用于获取知识。

**国产替代：秘塔 AI 搜索 / 360 AI / 腾讯元宝**
- **秘塔**：以无广告、学术搜索强著称，非常适合写论文查资料。
- **腾讯元宝**：深度整合公众号生态，能搜到很多独家优质文章。

### 8.5.3 编程与代码辅助 (Coding)

即使你不是程序员，也可以用它们写简单的脚本或网页。

**国际首选：Cursor / GitHub Copilot**
- **Cursor**：一款自带 AI 的编辑器，你可以用自然语言（中文）指挥它写代码，"Tab"键一路狂按就能写出程序。

**国产替代：通义灵码 / DeepSeek Coder**
- **通义灵码**：在国内 IDE 中渗透率极高，免费且速度快。
- **DeepSeek**：性价比极高，很多开发者用它的 API 配合开源插件使用。

### 8.5.4 音视频创作 (Audio & Video)

**视频生成：**
- **Sora (OpenAI)**：视频生成的标杆，物理规律准确。
- **可灵 (Kling) / 即梦 (Jimeng)**：国产之光。生成速度快，甚至支持生成 1080p 长视频，在某些人像场景下超越 Sora。

**音乐生成：**
- **Suno / Udio**：输入歌词生成歌曲，效果堪比专业录音室。
- **天工 SkyMusic**：国产音乐大模型，中文咬字更清晰，适合中文歌曲创作。

### 8.5.5 工具箱推荐总结表

| 场景 | 🌍 国际首选 | 🇨🇳 国产推荐 |
| :--- | :--- | :--- |
| **笔记文档** | Notion AI | 飞书智能伙伴, WPS AI |
| **深度搜索** | Perplexity | 秘塔 AI, 腾讯元宝 |
| **写代码** | Cursor | 通义灵码 |
| **画图** | Midjourney | 可灵, 通义万相 |
| **做视频** | Sora | 可灵, 即梦 |
| **做音乐** | Suno | 天工 SkyMusic |
| **PPT制作** | Gamma | WPS AI, 讯飞智文 |

**小建议**：
工具在精不在多。对于国内用户，**"WPS AI (文档) + 秘塔 (搜索) + 可灵 (视频) + 通义/Kimi (对话)"** 是一套非常黄金的 2026 年效能组合。
