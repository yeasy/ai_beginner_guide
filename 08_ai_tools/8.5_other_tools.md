# 第八章：主流 AI 工具使用指南

## 8.5 其他实用 AI 工具

除了前面介绍的通用对话 AI 工具，在特定领域还有很多身怀绝技的专业工具。这里挑选几个最适合普通人上手、好玩又实用的。

### 8.5.1 给文章配个图：Midjourney & 可灵

不想从网上盗图，想拥有独一无二的插画？

**Midjourney (画图)**
- **特点**：目前公认艺术感最强、画面最精美的 AI 绘画工具。
- **怎么用**：目前主要在 Discord（一个聊天软件）里使用。输入 `/imagine` 加上你的描述（比如 "一只戴着宇航员头盔的猫，赛博朋克风格"），它就生成 4 张图供你挑。
- **适合**：做头像、文章配图、设计灵感、海报背景。

**可灵/即梦 (国产之光)**
- **特点**：国内快手和字节跳动推出的工具，中文支持极好，还能生成视频。
- **怎么用**：直接用手机 App 或网页，输入中文描述即可。
- **适合**：生成动图、短视频素材、不想折腾网络环境的用户。

### 8.5.2 还是不懂怎么搜：Perplexity

这可能是比百度/谷歌更好用的"下一代搜索引擎"。

- **痛点**：传统搜索给你十个链接，你要一个个点进去看，还得忍受广告。
- **Perplexity 做法**：你问问题，它直接阅读搜索结果，然后给你写出一个**带有引用来源的总结答案**。
- **怎么用**：打开 `perplexity.ai`，把它当成"懂搜索的 ChatGPT"用。问它"2026 年最新款的手机哪款拍照最好？"，它会综合各大测评网站给你结论。

### 8.5.3 写歌做音乐：Suno & Udio

即使你五音不全，不懂乐理，现在也能写歌。

- **Suno**：音乐界的 ChatGPT。
- **怎么用**：输入歌词（或者只输入主题，如"一首关于加班太累想去海边度假的民谣"），点击生成。几十秒后，一首带人声演唱、编曲完整的歌就出来了。
- **效果**：惊人地好听，甚至能生成像模像样的流行金曲。大家可以在聚会时给朋友专门写一首歌助兴。

### 8.5.4 工具箱推荐总结

| 你的需求 | 推荐工具 |
| :--- | :--- |
| **综合问答、逻辑分析** | ChatGPT (OpenAI) |
| **写情感细腻的文章、读长书** | Claude (Anthropic) |
| **处理视频、Google生态用户** | Gemini (Google) |
| **国产通用工具** | 通义千问、豆包、文心一言（详见 8.4 节） |
| **搜集最新确切信息** | Perplexity |
| **画图、做海报** | Midjourney, 可灵, 通义万相 |
| **写歌娱乐** | Suno |
| **读中文长文档** | Kimi（详见 8.4.4 节） |
| **代码与推理** | DeepSeek（详见 8.4.3 节） |

大家不必贪多，先挑一个最顺手的深度使用，习惯了再探索其他的。记住，工具本身不重要，**解决你的问题**才重要。
