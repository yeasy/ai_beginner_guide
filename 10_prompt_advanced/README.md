# 第十章：复杂任务与深度推理

在掌握了基础提示词技巧后，我们开始挑战更高难度的任务。

本章将探讨如何引导 AI 进行深度思考，解决复杂的逻辑、数学和推理问题，以及如何像产品经理一样迭代优化你的 Prompt。

**本章内容速览：**

*   **[10.1 思维链（CoT）](10.1_chain_of_thought.md)**：让 AI 学会"慢思考"，展示推理步骤，大幅提升准确率。
*   **[10.2 迭代优化](10.3_iterative.md)**：好的提示词不是一次写成的，而是"试"出来的。学习迭代优化的心法。
*   **[10.3 高级技巧](10.4_advanced_techniques.md)**：自洽性、角色反转、分治法等解决极难任务的"黑科技"。
